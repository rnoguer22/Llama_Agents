JUST AFTER THE INSTALATION

--> No se podia arrancar desde windows, tuve que habilitar el dual boot instalando grub
    sudo nano /etc/default/grub
    Cambiar las siguientes lineas dentro del archivo:
    --> GRUB_TIMEOUT_STYLE="hidden" --> "menu"
    --> GRUB_TIMEOUT="0" --> "10"

--> --> Cuando aparecia el dual boot, salia demasiado grande, lo tuve que poner + pequeño
	sudo nano /etc/default/grub
	dentro del fichero grub, añadir las siguientes lineas:
	GRUB_GFXMODE="1024x768"
	GRUB_GFXPAYLOAD_LINUX="keep"
	Probe con muchas mas resoluciones, pero ninguna otra + pequeña la soporta la tarjeta grafica

--> Cambiar tipografia de vscode
    Siempre me ha dado problemas cambiar la fuente de vscode, pero esta vez lo consegui
    --> 1) Instalar la fuente que queramos usar, Cascadia Code en este caso
    --> 2) Descomprimir el archivo y mover las carpetas 
	Al descomprimir, obteniamos un archivo zip que al descomprimirlo resulta en las carpetas otf, ttf, woff2
	Las movemos a la ruta donde debemos tener nuestras propias fuentes:
	--> mkdir -p ~/.local/share/fonts
	--> mv otf ttf woff2 ~/.local/share/fonts
	Actualizar cache de las fuentes
	--> fc-cache -f -v
   --> 3) Cambiar el archivo de configuracion de vscode para añadir la fuente:
	--> sudo nano ~/.config/Code/User/settings.json
	Añadir las siguientes lineas: 
	--> "editor.fontFamily": "Cascadia Code, Consolas, 'Courier New', monospace"
	--> "editor.fontLigatures": true,
  	--> "editor.fontWeight": "normal"
	En este caso añadimos las ligaduras (ya que esta fuente las soporta)
